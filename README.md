# 🌸 ButterBloom Cafe Website
*🌼 where every bite blooms with flavor. 🌼*

#### A cozy and elegant cafe website built using HTML, CSS and JavaScript, inspired by modern bakery and coffee shop designs.

#### ButterBloom offers an aesthetic browsing experience with interactive menus, category filters, smooth navigation and responsive layouts.
## ✨ Features 

#### Beautiful cafe inspired UI
#### Responsive navigation bar
#### Interactive hero section
#### Categorized menu section
#### Active category highlighting
#### Dynamic menu filtering using javascript
#### Dedicated orders page
#### Contact section with embedded map
#### Smooth scrolling navigation

## 🛠️ Techstack Used

#### HTML
#### CSS
#### JavaScript
#### Git & GitHub

## 🚀 Functionalities

#### Menu Filtering
#### Users can switch between categories using interactive buttons
#### Selected categories remain highlighted
#### Only the chosen food section is displayed
